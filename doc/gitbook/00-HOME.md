# MBGUI

Bem-vindo a documentacao do projeto MBGUI.

Esta base existe para concentrar o conhecimento essencial do sistema em um lugar facil de navegar, revisar e manter. A ideia e reduzir dependencia de conhecimento oral e acelerar desenvolvimento, validacao e suporte.

## Para quem e esta documentacao

- Desenvolvedores que precisam entender a base de codigo
- QA que precisa validar fluxos e reproduzir bugs
- Lideranca tecnica que precisa acompanhar riscos e decisoes
- Suporte interno que precisa consultar comportamento esperado
