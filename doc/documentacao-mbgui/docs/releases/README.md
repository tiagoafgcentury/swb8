# Releases do MBGUI

Esta pasta concentra notas de release, histórico de versões e consolidação de mudanças relevantes do MBGUI.

## Exemplos de documentos para esta pasta

- Release 1.3.2
- Release 1.3.3
- Consolidado de bugs corrigidos por versão
- Riscos conhecidos por release

## Convenção de nomes

Padrões sugeridos:

- `release-<versao>.md`
- `release-<versao>-bugs.md`
- `release-<versao>-riscos.md`

Exemplos:

- `release-1.3.2.md`
- `release-1.3.2-bugs.md`
- `release-1.3.2-riscos.md`

## Regra de uso

Crie novos arquivos `.md` nesta pasta quando o objetivo for documentar entregas, versões, mudanças publicadas e acompanhamento por release.
