# Processos do MBGUI

Esta pasta concentra processos internos e fluxos de trabalho do time relacionados ao MBGUI.

## Exemplos de documentos para esta pasta

- Processo de homologação
- Processo de triagem de bugs
- Processo de preparacao de release
- Processo de investigação em bancada

## Convenção de nomes

Padrão sugerido:

- `NN-NOME-DO-PROCESSO.md`

Exemplos:

- `01-STANDBY-RECEPTOR.md`
- `02-INSTALA-FACIL-RECEPTOR.md`
- `03-ATUALIZACAO-OTA.md`
