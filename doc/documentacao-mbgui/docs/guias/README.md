# Guias do MBGUI

Esta pasta concentra guias práticos, materiais de onboarding e checklists operacionais do MBGUI.

## Exemplos de documentos para esta pasta

- Guia de onboarding técnico
- Guia de análise de bugs
- Guia de checklist antes de release
- Guia de validação de instalação fácil

## Convenção de nomes

Padrões sugeridos:

- `guia_<tema>.md`
- `checklist_<tema>.md`
- `onboarding_<tema>.md`

Exemplos:

- `guia_configuracao_rede.md`
- `checklist_validacao_instalacao_facil.md`
- `onboarding_desenvolvedor_mbgui.md`

## Regra de uso

Crie novos arquivos `.md` nesta pasta quando o objetivo for orientar pessoas em um fluxo prático, passo a passo ou recorrente.
