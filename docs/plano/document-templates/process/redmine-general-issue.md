# Template Redmine: demanda geral

Use para tarefas, melhorias, documentacao ou ajustes tecnicos que nao sejam bug de QA.

## Titulo

```text
[Tipo] Resumo objetivo da demanda
```

Exemplos:

```text
[Docs] Documentar build release do B8
[Task] Revisar fluxo de atualizacao USB
[Feature] Ajustar regras de lista de canais Claro
```
