# Template Redmine: bug reportado por QA/teste

Use quando QA, suporte ou teste de bancada abrir um bug.

## Titulo

```text
[Bug] <modulo/fluxo>: <resumo do problema>
```

Exemplo:

```text
[Bug] Instala Facil: receptor monta lista de canais ao sair do stand-by
```
