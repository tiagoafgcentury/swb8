# Portal do Plano de Documentacao B8

Este diretorio contem um projeto MkDocs para publicar os arquivos Markdown oficiais do plano de documentacao.

O portal e gerado diretamente a partir de:

```text
docs/plano/
```

Nao ha copia intermediaria dos documentos em `docs/ppt-plano/`.

## Como gerar

Execute:

```bat
script.cmd
```

O site sera gerado em:

```text
docs/ppt-plano/site/
```

Pagina inicial gerada:

```text
docs/ppt-plano/site/index.html
```

## Observacao

Edite os documentos oficiais em `docs/plano/`. O diretorio `docs/ppt-plano/` deve conter apenas a configuracao do portal, ferramentas auxiliares e o site gerado.
